HTML Semantic Tags

Semantic tags in HTML provide meaning to the content of a page, improving accessibility and SEO. These tags describe the role of the content they contain, making the webpage more understandable for search engines and screen readers.

Example of Semantic Tags:

<header>Header content</header>
<nav>Navigation links</nav>
<main>Main content of the page</main>
<footer>Footer content</footer>

Key Semantic Tags:

  • <header>: Represents introductory content or navigational links.
  • <nav>: Defines navigation links.
  • <main>: Represents the dominant content of the document.
  • <footer>: Contains footer information about the document.

CSS Functions

CSS functions are used to perform operations, modify styles, or handle dynamic behavior on a webpage. These functions enable you to do things like manipulating colors, calculating dimensions, and applying transformations.

Example of CSS Functions:

background-color: rgb(255, 0, 0); /* RGB color function */
width: calc(100% - 20px); /* Calc function for calculation */
transform: rotate(45deg); /* Rotate function */

Common CSS Functions:

  • rgb() / rgba(): Defines colors using Red, Green, and Blue (RGB) with optional Alpha (A) for transparency.
  • calc(): Performs calculations to set property values (e.g., width, height).
  • rotate(): Rotates an element by a specified degree.
  • scale(): Resizes an element (e.g., scaling it up or down).
  • translate(): Moves an element along the X or Y axis.
